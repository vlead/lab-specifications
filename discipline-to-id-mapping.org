* Introduction
  This document maps a discipline id to a discipline name. 

* Assignment
  Rules for assigning a discipline ID to a discipline
  1. DisciplineId = Acronym of the discipline name
  2. The discipline ID doesn't contain any hyphens
  3. For a new discipline, the discipline ID is decided
     first before giving any new lab IDs or updating any
     other assets in the database.
