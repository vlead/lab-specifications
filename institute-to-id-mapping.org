* Introduction
  This document maps an institute id to an Institute name. 

* Assignment
  Rules for assigning a institute ID to a institute
  1. Institute Id is the Acronym of the institute name
  2. The Institute ID doesn't contain any hyphens
  3. For a new institute, the institute Id is decided
     first before giving any new lab IDs or updating any
     other assets in the database.
